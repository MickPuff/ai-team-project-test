# Roadmap

## Phase 0: Discovery Preparation

Goal: Prepare repo, team roles, templates, and intake structure before May 20, 2026 fieldwork.

Deliverables:

- Discovery plan
- Stakeholder interview guide
- Data inventory template
- Pain point log
- Reporting requirements template
- Team ownership map

## Phase 1: Discovery Synthesis

Goal: Convert gathered business/data/reporting inputs into clear product requirements.

Deliverables:

- Current-state workflow map
- Pain point summary
- Source data inventory
- Dashboard/report catalog
- MVP recommendation

## Phase 2: Data Foundation

Goal: Define and implement the first reliable data model and import path.

Deliverables:

- Database schema
- Data dictionary
- Spreadsheet import format
- Validation rules
- Seed/sample data

## Phase 3: Core Back Office Workflow

Goal: Build the first usable workflow for room occupancy, income, and expense tracking.

Deliverables:

- Room registry
- Occupancy updates
- Income records
- Expense records
- Basic role-aware navigation

## Phase 4: Dashboard And Reports

Goal: Provide summary views for owner, manager, finance, and operations users.

Deliverables:

- Occupancy dashboard
- Income/expense dashboard
- Monthly report
- Exportable summaries

## Phase 5: Hardening And Handoff

Goal: Validate with users, improve reliability, and prepare for production use.

Deliverables:

- UAT fixes
- Permissions
- Backup/export plan
- Operations documentation
